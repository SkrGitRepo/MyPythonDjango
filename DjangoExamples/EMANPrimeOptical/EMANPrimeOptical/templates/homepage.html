EMAN Onramp - Cisco Prime Optical user provisioning report

This is a report tool developed to view all Cisco Prime Optical (CPO) users, provisioned from EMAN-OnRamp tool. from here you can view report for CPO users provisioned from EMAN-Onramp based on specific CPO User-Role for which respective Onramp resources(dev/prod) are available on EMAN-Onramp side for subscription.

Cisco Prime Optical users are categorized into bellow user-roles:

  1. Superuser
  2. System Admin
  3. Operator
  4. Provisioner
  5. Network Admin
